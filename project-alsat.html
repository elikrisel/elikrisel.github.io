Project Info

Role: Game Programmer
Team Size: 8
Engine: Unity Engine

Introduction

This was my first big project, we got assigned to make a puzzle game and after some brainstorming we decided to go for a co-operative platformer with the main mechanic where the players have to teleport each other to get through the levels and solve different puzzles.

Code

I focused most of my time on working on the movement system and also implementing the Health system for both of the characters. I also worked on other game mechanics such as Platform mechanics and checkpoint system.

Noteworthy Achievements

Third place in Class choice

Conclusions

This was my introduction to make a game from scratch, all of us did not have much experience in making games. We planned too big from the start and it was a good lesson moving forward. I got along well with all my members and had a great co-operation with my other programmer.
